### **Deep Link Handler (`DeepLinkHandler.component.tsx`)**

```typescript
import { App as CapacitorApp } from '@capacitor/app';
import { useRouter } from 'next/navigation';

export function DeepLinkHandler() {
  const router = useRouter();

  useEffect(() => {
    CapacitorApp.addListener('appUrlOpen', (event) => {
      const url = new URL(event.url);
      const path = url.pathname;
      router.push(path);
    });
  }, [router]);

  return null;
}
```

**How it works:**
1. User taps `capo://join/berkotnf/abc123`
2. iOS/Android opens app
3. Capacitor fires `appUrlOpen` event
4. Handler extracts path: `/join/berkotnf/abc123`
5. Next.js router navigates to existing page component

## ğŸ“± **API Communication**

### **Mobile API Helper** (`src/lib/apiConfig.ts`)

**Purpose:** Automatically route API calls to correct endpoint

```typescript
export function getApiBaseUrl(): string {
  // Capacitor app in production
  if (Capacitor.isNativePlatform() && process.env.NODE_ENV === 'production') {
    return 'https://app.caposport.com/api';
  }
  
  // Development mode (web or mobile)
  if (process.env.NODE_ENV === 'development') {
    return 'http://localhost:3000/api';
  }
  
  // Web app (relative paths)
  return '/api';
}

export async function apiFetch(path: string, options?: RequestInit) {
  const url = `${getApiBaseUrl()}${path}`;
  return fetch(url, {
    ...options,
    credentials: 'include',
    headers: {
      'Content-Type': 'application/json',
      ...options?.headers,
    },
  });
}
```

**Usage:**
```typescript
// Everywhere in the app
import { apiFetch } from '@/lib/apiConfig';
const response = await apiFetch('/players');
```

**Automatic routing:**
- Web dev: `http://localhost:3000/api/players`
- Web prod: `/api/players` (same origin)
- Mobile dev: `http://localhost:3000/api/players` (via live reload)
- Mobile prod: `https://app.caposport.com/api/players`

## ğŸ”‘ **Authentication**

**Method:** Supabase Phone Authentication (all users)

**Flow:**
1. User enters phone number
2. Supabase sends SMS OTP via Twilio
3. User enters 6-digit code
4. Supabase creates session (JWT token)
5. Token stored securely (iOS Keychain / Android Keystore via Capacitor)
6. Session auto-refreshes

**Mobile-specific:**
- Cookies use `SameSite=None; Secure` for cross-origin
- API calls include `credentials: 'include'` (via `apiFetch()`)
- Session persists across app restarts

**See:** `SPEC_auth.md` v6.6 for complete auth architecture
